# 🌙 Lunex Browser

A modern, privacy-focused web browser built with speed, simplicity and customization in mind.

> **Current Version:** v0.3 *(Early Development)*

---

## ✨ About

Lunex is a lightweight browser designed to provide a clean and intuitive browsing experience while putting privacy first. The goal is to combine a modern interface with powerful privacy features and extensive customization without sacrificing performance.

Although still in active development, Lunex is already usable for everyday browsing and continues to improve with every release.

---

## 🚀 Features

* 🔒 Privacy-focused browsing
* 🕶️ Private Mode with RAM-only sessions
* 🧅 Integrated Tor browsing
* ⚙️ Highly customizable interface
* 🎨 Modern, minimal UI
* 📑 Fast tab management
* 🔍 Multiple search engine support
* 🛡️ Improved security architecture

## 📌 Project Goals

Lunex aims to become a browser that is:

* Fast
* Secure
* Private
* Minimalistic
* Beautiful
* Highly customizable

Instead of adding unnecessary features, Lunex focuses on creating a clean browsing experience that simply feels good to use.

---

## ⚠️ Current Status

Lunex is currently in **early development**.

Most core features are already working, but some parts are still being refined and improved. Bugs, missing features and UI changes are expected while development continues.
